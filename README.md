# Cimex-Booking

Cimex-Booking is an AI-assisted ASP.NET Core MVC project for a solo extermination business. It will provide a customer booking flow and an internal admin portal, while being designed to evolve into a multi-tenant SaaS platform.

## Planned Stack

- .NET 8 ASP.NET Core MVC
- ASP.NET Core Identity (admin-only access)
- Entity Framework Core
- PostgreSQL
- AWS Elastic Beanstalk

## Milestones

- **M1**: Repository and architecture scaffolding
- **M2**: Core domain and persistence foundation
- **M3**: Admin authentication and authorization baseline
- **M4**: Booking intake workflow (customer-facing)
- **M5**: Admin operations portal
- **M6**: Deployment and operational hardening
- **M7**: SaaS readiness and tenant boundaries
